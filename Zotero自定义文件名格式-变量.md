# Zotero 条目类型与字段

本页面提供 Zotero 条目类型和字段的通用描述。在手动向 Zotero 输入条目数据时，这些信息会很有帮助。本页面还就不常见条目类型的信息存储方式提供指导。

## 条目字段变量
### 通用字段变量
| {{ itemType }}        | **条目类型**   | `Item Type`                                 | 父条目类型。条目的广义类型。条目类型通常根据条目应如何被引用来确定。请参阅上文对 Zotero 条目类型的描述。您可以在类型(`Type`)、格式(`Format`)或其他(`Extra`)字段中，或使用 [标签]，来存储关于条目类型的更具体信息（例如，是小说还是传记类图书）。 |
| {{ title }}           | **标题**       | `Title` `Case Name` `Name Of Act` `subject` | 父条目的标题。应以句首字母大写的方式输入。 <br> 包括以下字段：<br> `Title`其他条目标题；`Case Name` 司法案例名称；<br> `subject` 电子邮件主题；<br> `Name Of Act` 法律名称。 |
| {{ shortTitle }}      | **短标题**     | `Short Title`                               | 标题的简短形式，通常不带副标题。主要用于脚注风格中的后续引用。 |
| {{ attachmentTitle }} | 附件标题       | `attachmentTitle`                           | 正在被重命名或创建的附件的标题。 |
| {{ authors }}         | **主要创作者** | `authors`                                   | 父条目的主要创作者（取决于条目类型(`Item Type`)，这些是作者或艺术家，但不包括编辑或其他贡献者）。 |
| {{ authorsCount }}    | 主要创作者总数 | `authorsCount`                              | 父条目主要创作者的数量。（Zotero 7.0.16） |
| {{ editors }}         | **编辑**       | `editors` `Series Editors`                  | 父条目的编辑(`editors`)和丛书编辑(`Series Editors`)字段。 <br> 包括以下字段：<br> `editors` 图书编辑、图书章节编辑、会议论文编辑、词条编辑、百科条目编辑、报告编辑、文档编辑、期刊文章编辑、标准编辑、预印本编辑；<br> `Series Editors` 图书丛书编辑、图书章节丛书编辑、会议论文丛书编辑、词条丛书编辑、百科条目丛书编辑、报告丛书编辑、地图丛书编辑。 |
| {{ editorsCount }}    | 编辑总数       | `editorsCount`                              | 父条目编辑和丛书编辑的数量。（Zotero 7.0.16） |
| {{ creators }}        | **创作者**     | `Creators`                                  | 父条目的所有创作者，与作品相关的个人或实体。一个条目可以有多个不同类型的创建者。请参阅下面的创建者类型列表。 |
| {{ creatorsCount }}   | 创作者总数     | `creatorsCount`                             | 父条目所有创作者的总数。（Zotero 7.0.16） |
| {{ firstCreator }}    | 创作者列值     | `firstCreator`                              | 父条目的创作者（1-2 位作者或编辑），与主面板中的“创作者”列的值相同。 |
| {{ abstractNote }}    | **摘要**       | `Abstract`                                  | 父条目的简短描述。 |
| {{ language }}        | **语言**       | `Language`                                  | 父条目的出版语言。软件使用编程语言(`Prog.Language`)。我们建议将其存储为两个字母的[ISO语言代码]，后跟两个字母的[ISO国家/地区代码]（例如，en-US 表示美式英语，de-DE 表示德语）。语言(`Language`)字段用于确定在格式化条目标题时是否应应用[英文标题大小写规则]。如果存储的条目数据是英文的（例如，对于翻译后的标题），即使条目内容是另一种语言，也应输入 en-US 或 en-UK。 |
| {{ date }}            | **日期**       | `Date`                                      | 父条目出版日期。对于电子资源的访问日期，使用“访问日期(`Accessed`)”。 |
| {{ year }}            | 出版年份       | `year`                                      | 父条目出版年份。从父条目的日期(`Date`)字段中提取。 |
| {{ citationKey }}     | **引用关键词** | `Citation Key`                              | 引用关键词字段。 |

| {{ bookTitle }}           | **书名**     | `Book Title`                                                                                                                                               | 图书章节所在的图书的标题。 |
| {{ dictionaryTitle }}     | **词典标题** | `Dictionary Title`                                                                                                                                         | 词条所在的词典的标题。 |

| {{ pages }}               | **页码**     | `Pages` `Code Pages` `First Page`                                                                                                                          | 作为较大出版物一部分的条目的页码范围。也用于电子期刊和电子图书中的定位符或文章编号。 <br> 包括以下字段：<br> `Pages` 图书章节页码、会议论文页码、词条页码、百科条目页码、听证会页码、期刊文章页码、杂志文章页码、报纸文章页码、专利页码、报告页码、法律页码；<br> `Code Pages` 法案法典页码；<br> `First Page` 司法案例起始页。 |
| {{ volume }}              | **卷次**     | `Volume` `Code Volume` `Reporter Volume`                                                                                                                   | 文章或章节所在的（通常是多卷）出版物的卷号。对于数字卷号，仅输入数字。如果引用时需要同时包含卷标题，请在此字段中包含标题（例如，“2：组织心理学”）。 <br> 包括以下字段：<br> `Volume` 音频卷次、图书卷次、图书章节卷次、会议论文卷次、词条卷次、百科条目卷次、期刊文章卷次、杂志文章卷次、报纸文章卷次、视频卷次；<br> `Code Volume` 法案法典卷次；<br> `Reporter Volume` 司法案例报告系统卷次。 |
| {{ numPages }}            | **总页数**   | `# of Pages`                                                                                                                                               | 图书的总页数。 <br> `# of Pages` 图书总页数、手稿总页数、标准总页数、学位论文总页数。 |
| {{ numberOfVolumes }}     | **总卷数**   | `# of Volumes`                                                                                                                                             | 集体作品的总卷数。 <br> `# of Volumes` 音频总卷数、图书总卷数、图书章节总卷数、会议论文总卷数、词条总卷数、百科条目总卷数、听证会总卷数、视频总卷数。 |
| {{ series }}              | **系列**     | `Series`                                                                                                                                                   | 包含多个出版物或演示文稿的丛书名称（例如，“剑桥比较政治学研究”，“人格评估进展特刊”）。 <br> `Series` 图书系列、图书章节系列、会议论文系列、词条系列、百科条目系列、期刊文章系列、演示文档系列、学位论文系列、预印本系列。 |
| {{ seriesTitle }}         | **系列标题** | `Series Title`                                                                                                                                             | 期刊一期中一系列文章的标题（例如，一个专题部分或“封面故事”）。将其称为“部分标题”可能更合适。有关说明，请参见[此处][系列标题]。 <br> `Series Title` 音频系列标题、期刊文章系列标题、地图系列标题、播客系列标题、报告系列标题、软件系列标题、视频系列标题。 |
| {{ seriesNumber }}        | **系列编号** | `Series Number`                                                                                                                                            | 条目在丛书中的编号。 <br> `Series Number` 图书系列编号、图书章节系列编号、会议论文系列编号、词条系列编号、百科条目系列编号、报告系列编号、学位论文系列编号、预印本系列编号。 |
| {{ seriesText }}          | **系列描述** | `Series Text`                                                                                                                                              | 期刊文章系列描述。 |
| {{ journalAbbreviation }} | **刊名简称** | `Journal Abbr`                                                                                                                                             | 期刊的缩写标题。使用 Zotero Word 或 LibreOffice 插件时，Zotero 可以使用 MEDLINE 列表自动缩写标题。使用此字段输入基于其他系统的缩写，或在文字处理器之外生成参考文献目录。 |
| {{ edition }}             | **版本**     | `Edition`                                                                                                                                                  | 作品的版本。如果是数字，请简化为阿拉伯数字（即，输入“2”而不是“second”）。 <br> `Edition` 图书版本、图书章节版本、词条版本、百科条目版本、地图版本、报纸文章版本、标准版本。 |

| {{ section }}             | **条文序号** | `Section`                                                                                                                                                  | 期刊和报纸的版次或部分。 <br> `Section` 期刊文章条文序号、报纸文章条文序号、法案条文序号、法律条文序号。 |

### 与条目访问时间和方式相关的字段变量
| {{ DOI }}             | **DOI**      | `DOI`             | 条目的[数字对象标识符]。对于期刊文章以外的条目，DOI 可以存储在其他(`Extra`)字段中。请参阅下面的从其他(`Extra`)字段引用。 |

## 条目类型

Zotero 中的条目类型应被视为灵活的、宽泛的类别。条目类型通常根据条目应如何被引用来确定。请参阅下文对 Zotero 条目类型的描述。如果您要引用一个不完全符合任何现有类别的不常见条目类型，请选择一个相对接近且具有足够字段的条目类型（例如，**图书**、**报告**或**手稿**可以很好地适配大多数不常见的条目类型）。

您可以在**类型(`Type`)**、**格式(`Format`)**或**其他(`Extra`)**字段中，或使用标签，来存储关于条目类型的更具体信息（例如，是小说还是传记类图书）。

| 条目类型-中文 | 条目类型-英文 | 描述 |
| :------------ | :------------ | :--- |
| **艺术品**   | `Artwork`              | 一件艺术品（例如，油画、照片或雕塑）。此条目类型也适用于其他类型的图像或视觉项目（例如，科学图表）。 |
| **附件**     | `Attachment`           | 独立的附件文件（例如，PDF、JPEG、DOCX、PPTX、XLSX 或 ODT 文件）。独立的附件文件在 Zotero 中功能有限（例如，无法正确搜索或引用）。始终将文件附加到正确的 Zotero 条目上。 |
| **音频**     | `Audio Recording`      | 任何形式的音频录制，包括音乐、有声读物、音效、档案录音或基于音频的科学图表。 |

| **博客帖子** | `Blog Post`            | 发布在个人博客网站上的文章或帖子。对于作为大型在线出版物一部分发布的在线文章（例如，纽约时报博客），使用**杂志文章**或**报纸文章**通常效果更好。 |
| **图书章节** | `Book Section`         | 一本书的一个部分。通常是章节，但也包括前言、序言、引言、附录、后记、评论等。 |

| **作者\***        | Words By*       | 在音频录制中撰写文字的个人（例如，词作者或演讲稿撰写人）。 |
| **编剧\***        | Scriptwriter*   | 为电影或广播撰写剧本的人。 |
| **宾客\***        | Guest*          | 播客、无线电广播或电视广播的嘉宾。 |
| **顾问\***        | Counsel*        | 为法律判例辩护的法律顾问。 |
| **贡献者\***      | Contributor*    | 与作品相关的其他个人或实体，不应包含在引文中（例如，图书封面上“with”后面列出的人员）。用作**听证会**条目的**作者**。 |
| **评论者\***      | Commenter*      | 对博客文章发表评论的个人。 |
| **制片人\***      | Producer*       | 电影、录制或广播的制片人。 |
| **演员阵容\***    | Cast Member*    | 广播或表演的演员阵容成员。 |
| **共同发起人\***  | Cosponsor*      | 法案或立法的共同提案人或支持者。 |
| **代理师/机构\*** | Attorney/Agent* | 在申请专利时代表发明人的律师或代理人。 |

### 媒体创作者的角色标签
对于电影、录制和广播的引用，Zotero 目前对标注制片人、编剧和其他一些创建者角色的支持有限。

要标注导演，请将主要的 Zotero 字段留空（或将姓名作为**贡献者**输入），并使用**其他(`Extra`)**字段中的**导演**输入导演姓名。请参阅下面的从[其他(`Extra`)字段引用]。

所有创建者角色（**导演**、**制片人**、**编剧**等）也可以通过以下方式标注：使用条目的默认作者角色（**音频录制**为**表演者**，**播客**为**播客主持**，**电影**、**电台广播**、**电视广播**和**视频**为**导演**）输入姓名，并根据您的引用风格在作者的名字或姓氏后添加适当的标签（括号内）——例如，对于 APA 风格，输入 `MacNaughton || Ian (Producer)`。请注意，标签将在引文中逐字呈现；根据需要在此处输入缩写术语（例如，“Prod.”）。

如果引用风格使用作者名字的首字母缩写而不是全名（例如 APA 风格），并且标签包含多个单词（例如，“Executive Producer”或“Writer & Director”），Zotero 会缩写标签中第一个单词之后的单词。为避免这种情况，请在标签中每个空格的两侧输入一个“单词连接符”字符（Unicode U+2060，此处打印在引号之间：“⁠”）。

另请参阅[媒体创作者角色](https://www.zotero.org/support/kb/media_creator_roles)

## 其他条目类型和字段
### Zotero 中未包含的可引用条目类型
Zotero 目前尚未正式支持这些条目类型。出于引用目的，您可以通过将它们以以下格式输入到**其他(`Extra`)**字段中，将其他类型的条目转换为这些类型：

`Type: CSL Type`

例如：
`Type: review-book`

| 条目类型 | CSL 类型 | 描述 |
| :------- | :------- | :--- |
| 图表   | `figure`        | 包含在科学或学术作品中的图表。 |
| 乐谱   | `musical_score` | 音乐作品的书面乐谱。 |
| 小册子 | `pamphlet`      | 非正式出版的作品。通常比**报告**篇幅更小，技术性更低。 |
| 书评   | `review-book`   | 对一本书的评论。根据发表位置，将这些条目作为**期刊文章**、**杂志文章**或**报纸文章**输入，并提供“**被评论作者**”创建者。 |
| 条约   | `treaty`        | 两国之间的法律条约。 |

### Zotero 中未包含的可引用字段
Zotero 目前尚未正式支持这些条目字段。出于引用目的，您可以通过将它们以以下格式输入到**其他(`Extra`)**字段中，将其他类型的条目转换为这些类型：

`CSL Variable: Value`

例如：
`PMID: 123456`
`Status: in press`
`Original Date: 1886-04-01`
`Director: Kubrick || Stanley`

| 字段 | CSL 变量 | 描述 |
| :--- | :------- | :--- |
| PMID         | `PMID`                     | PubMed 标识符。 |
| PMCID        | `PMCID`                    | PubMed Central 标识符。 |
| 状态         | `Status`                   | 条目的出版状态（例如，“即将出版”、“印刷中”、“在线提前出版”）。 |
| 提交日期     | `Submitted`                | 条目提交出版的日期。 |
| 被评论标题   | `Reviewed Title`           | 被评论作品的标题。 |
| 章号         | `Chapter Number`           | 图书内的章节编号。 |
| 档案馆地点   | `Archive Place`            | 档案馆的地理位置。 |
| 事件日期     | `Event Date`               | 事件发生的日期。以 ISO 格式输入（年-月-日）。 |
| 事件地点     | `Event Place`              | 事件的地理位置。 |
| 原始日期     | `Original Date`            | 条目最初出版的日期。以 ISO 格式输入（年-月-日）。 |
| 原始标题     | `Original Title`           | 作品的原始标题（例如，未翻译的标题）。 |
| 原始出版社   | `Original Publisher`       | 条目原始版本的出版者（例如，未翻译版本的出版者）。 |

| 原始作者     | `Original Author`          | 一种创建者类型。作品的原始创作者。 |
| 导演         | `Director`                 | 一种创建者类型。电影、录制或广播的导演。在 Zotero 中，“Director”映射到 CSL author。如果您需要为导演添加特殊标签——“(Dir.)”，请在**其他(`Extra`)**字段中输入导演标签。 |
| 编辑总监     | `Editorial Director`       | 一种创建者类型。出版物的执行编辑。 |
| 插图师       | `Illustrator`              | 一种创建者类型。作品的插图师。 |

### 从其他(`Extra`)字段引用
如果 Zotero 条目类型缺少引用所需的字段，可以将这些字段添加到**其他(`Extra`)**字段中。

在**其他(`Extra`)**字段顶部以以下格式将每个变量单独成行输入：

`CSL Variable: Value`

例如：
`DOI: 10.1128/AEM.02591-07`
`Original Date: 1824`
`PMCID: PMC3531190`

除了**条目类型**（CSL Type）和**日期**变量（CSL Issued 等）之外，在**其他(`Extra`)**字段中输入的变量不会覆盖在相应 Zotero 字段中输入的值。

**日期**
在**其他(`Extra`)**字段中输入的日期将覆盖在 Zotero 的**日期**字段中输入的日期。日期必须以 ISO 格式（年-月-日）输入。日期范围可以按以下格式输入：

`Issued: 2001-12-15/2001-12-31`

**姓名**
对于创建者变量，在**其他(`Extra`)**字段中用两个竖线字符 ('||') 分隔双字段姓名，如下所示：

`Editorial Director: De Gaulle || Charles`
